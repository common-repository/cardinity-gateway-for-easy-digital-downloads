== Installation ==

The integration process with Cardinity is smooth and easy. You can integrate by choosing one of the two ways.

= Install from WordPress Plugin Store: =

1. Navigate to your e-shop admin area.
2. Navigate to Plugins → Add New.
3. Type "Cardinity Payment Gateway for Easy Digital Downloads" into search bar.
4. Cardinity Plugin for Easy Digital Downloads should appear as a search result. Click "Install Now"
5. If installation was successful, Activate button should appear. Click the button.
6. Once the plugin is activated you can see it in Plugins list
7. Next step is plugin configuration. Navigate to Downloads→ Settings.
8. Click on Payment Gateways tab, activate Cardinity Gateway in Payment Gateways section and click Save Changes
9. Open Cardinity Gateway section, enter your test/live API keys and click Save Changes.
10. That’s it - you can start receiving Credit/Debit Card payment using Cardinity services!

= Install manually: =

1. Navigate to WordPress Plugin Website and download the latest version of plugin by clicking Download button
2. Sign-in into your e-shop member’s area.
3. Then navigate to Plugins → Add New.
4. Click on Upload Plugin link located at the top of the page. On the next windows click Choose File ( there you’ll need to choose Cardinity Easy Digital Downloads module, which you’ve downloaded from Cardinity website) and press Install Now.
5. If everything goes well you’ll see Activate Plugin link on the next page. This means that Cardinity payment module was successfully installed. 
6. Click the Activate Plugin link on the page you were redirected after module’s installation (see previous screenshot).
7. Once the plugin is activated you can see it in Plugins list
8. Next step is plugin configuration. Navigate to Downloads → Settings.
9. Click on Payment Gateways tab, activate Cardinity Gateway in Payment Gateways section and click Save Changes
10. Open Cardinity Gateway section, enter your test/live API keys and click Save Changes.
11. That’s it - you can start receiving Credit/Debit Card payment using Cardinity services!

SSL note: Cardinity requires that any page hosting a live checkout form be SSL (they should start with `https://`).
